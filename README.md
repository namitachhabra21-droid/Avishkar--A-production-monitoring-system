# Avishkar - Production Monitoring System

Avishkar is a production monitoring system built to track manufacturing tools, active production stages, machine-hour utilization, completion progress, and estimated timelines. The platform helps teams monitor shop-floor activity through role-based dashboards and structured tool update workflows.

## Features

- Role-based dashboards for production visibility
- Tool status tracking across manufacturing stages
- Active stage and completion percentage monitoring
- Planned vs completed machine-hour tracking
- Shift notes and production update workflows
- Estimated timeline and progress visibility
- MongoDB-backed production APIs
- Firebase integration for app-side workflows

## Tech Stack

- React Native
- Expo
- Node.js
- Express.js
- MongoDB
- Firebase
- JavaScript

## Project Overview

The system was designed to improve operational visibility in production workflows. It allows users to track the current status of tools, monitor stage-wise progress, record completed machine hours, and understand estimated completion timelines from a centralized dashboard.

- Integrated mobile frontend with backend APIs
